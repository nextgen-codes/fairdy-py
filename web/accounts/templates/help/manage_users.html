Block Cycle Limit

All new accounts have a Block Cycle Limit, which limits the amount of simulations the account can run on a one month rolling basis. Block Cycle Limit is calculated as the size of the storage system (in blocks) multiplied by the number of cycles. New accounts have a default Block Cycle Limit of 500 million. The default limit allows up to 100 simulations with up to 50,000 total blocks in each system, 100 cycles each. Enter -1 to give a user an unlimited block cycle limit. This is an integer field with a max. value of 2,147,483,647. Enter your choice in millions, it will be automatically multiplied by the web framework.

Edit User Dialog

To access the edit user dialog simply click the 'edit' button on the user record.

From within the dialog it is possible to modify the user's block cycle limit and 'is staff' setting.
