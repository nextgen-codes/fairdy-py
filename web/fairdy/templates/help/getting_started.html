Cellular Automata

A cellular automaton (pl. cellular automata, abbrev. CA) is a collection of boolean cells, also referred to as blocks in FaiRDy-Py, which together form a system. This system is then evolved over a set of discrete time cycles, with the state of a block in a given cycle determined by its neighbors' states in the previous cycle.
